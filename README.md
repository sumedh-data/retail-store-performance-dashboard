# Retail Store Performance Dashboard

## Project Overview

This project presents an interactive Retail Store Performance Dashboard developed in Power BI to analyze sales, profitability, product performance, and city-wise business performance. The dashboard enables decision-makers to monitor KPIs and identify trends through interactive visualizations and dynamic filtering.

## Objectives

* Monitor overall retail business performance
* Track revenue and profitability
* Analyze category-wise sales contribution
* Evaluate city-wise profit performance
* Identify monthly revenue trends
* Support data-driven decision-making

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

## Key Performance Indicators (KPIs)

* Total Revenue: 10.72M
* Total Profit: 1.58M
* Total Units Sold: 42K
* Profit Margin: 14.73%

## Dashboard Features

### Revenue Analysis

* Revenue distribution by product category
* Category contribution comparison

### Profit Analysis

* Profit by city
* Profit by product category
* Profit margin tracking

### Trend Analysis

* Monthly revenue trend visualization
* Performance monitoring across periods

### Interactive Filtering

* City-wise filtering
* Product category filtering
* Month-wise filtering

## Key Insights

* Beverages generated the highest revenue contribution.
* Solapur recorded the highest profit among all cities.
* Revenue showed noticeable fluctuations across months, highlighting seasonal trends.
* Product categories contributed relatively balanced revenue shares, reducing dependence on a single category.
